Section One

This is a quick experiment with Intersection observers which is a way for the viewport to detect elements that have come into view. We can use this instead of the scroll event listener, and comparing where the item is in comparison to the window. Not sure how to get it to interact with jQuery yet. This is a new ES6 method, and I am excited to learn more about it.
